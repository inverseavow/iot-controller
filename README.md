# IoT Controller

This project is designed to control Davinci devices and wallwashers via MQTT.

## Project Structure

- **backend/**: FastAPI application for controlling devices
- **frontend/**: React dashboard for monitoring and controlling IoT devices
- **mobile/**: React Native app for mobile device access
- **docs/**: Documentation for architecture and setup
- **.gitignore**: Files to ignore in the repository
- **docker-compose.yml**: Setup for containerized services
- **requirements.txt**: Python dependencies for the backend

## Setup

1. Clone the repository.
2. Set up your environment according to the documentation.
